From sum to product

You have a loop for calculating the sum of integer numbers in the range from start to end inclusively. The result is stored in the accumulator variable. Assume that the start and end variables are already declared as some integer values, so that start <= end.

int accumulator = 0;                   // (1)
for (int i = start; i <= end; i++) {   // (2)
    accumulator += i;                  // (3)
}

You need to change two lines in this code to calculate the multiplication of integer numbers in this range.
